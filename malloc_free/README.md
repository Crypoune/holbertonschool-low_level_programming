# C - Recursion

## Description

Ce projet introduit la gestion dynamique de la mémoire en langage C.
On y apprend à utiliser les fonction `malloc`, `free`, et à manipuler les pointeurs pour allouer et libérer de la mémoire manuellement.

## Contenu du projet

Les fichiers du projet incluent :

- `0-create_array.c` : crée un tableau de caractères et l'initialise avec un caractère spécifique.
- `1-strdup.c` : renvoie un pointeur vers une nouvelle chaîne de caractères qui est une copie de celle passée en argument.
- `2-str_concat.c` : concatène deux chaînes de caractères en une nouvelle chaîne.
- `3-alloc_grid.c` : crée une matrice d'entiers (tableau à deux dimensions).
- `4-free_grid.c` : libère la mémoire allouée à une grille d’entiers.
- `main.c` : (fichiers de test éventuels, non soumis à Holberton).

## Compilation

Toutes les fonctions doivent être compilées avec :

```bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 <main.c> <file.c> -o <output>
```
